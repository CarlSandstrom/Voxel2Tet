Required libraries
==================

 * VTK
 * hdf5
 	Install on Ubuntu: sudo apt-get install libhdf5-dev
 	Possibly use the FindHDF5.cmake file found at https://www.hdfgroup.org/ftp/HDF5/examples/examples-by-api/hdf5-examples-old/1_8/config/cmake/FindHDF5.cmake

 	if -lhdf5 does not work:
 		sudo ln -s /usr/lib/x86_64-linux-gnu/hdf5/serial/lib/libhdf5.so /usr/lib/libhdf5.so
 		sudo ln -s /usr/lib/x86_64-linux-gnu/hdf5/serial/lib/libhdf5_cpp.so /usr/lib/libhdf5_cpp.so

 * Armadillo
    Install on Ubuntu: sudo apt-get install libarmadillo-dev
